# mistelilab-tad-ge

### General Information

This is a repo containing the R code used to generate the results included in a bioRxiv preprint from the Misteli Lab at the National Cancer Institute/NIH. The manuscript explores the relationships between Gene Expression and Genome Architecture.

**Title:** TAD boundary architecture and gene activity are uncoupled.

**Authors:** Almansour F, Fursova NA, Keikhosravi A, Metz Reed KS, Larson DR, Pegoraro G, Misteli T

**DOI:** 10.64898/2025.12.13.694158

**PMID:** 41446082
